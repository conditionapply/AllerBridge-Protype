**AllerBridge — Travel hungry, not worried.**

AllerBridge is a safety-first travel food-allergy communication app designed for international travelers, study-abroad students, and families with food allergies.

Our prototype helps users create a personalized allergy profile, choose a travel destination, generate translated chef cards, review emergency phrases, and use a menu helper to spot obvious allergy concerns. Instead of claiming that food is “safe,” AllerBridge helps travelers ask the right questions about ingredients, sauces, shared oil, and cross-contact before ordering.

This video was created for the **2026 UDC Capital Cup Entrepreneurship Competition** by **Abhinav and Jackson**.

AllerBridge is not a medical device and does not guarantee allergen-free food. It is a communication and decision-support tool that helps travelers communicate more clearly and confidently when eating abroad.
